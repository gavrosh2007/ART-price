# ART-Sale — ПВА для торговли искусством

## Функции
- **Галерея** — общая лента всех работ
- **Книга** — авторы / коллекционеры
- Добавление работ (фото с телефона + ссылки)
- Кнопка "Покупаю" → сбор контактов
- Двуязычность (RU/EN)
- Размытый фон
- PVA (установка как приложения, офлайн-режим)

## Установка на GitHub Pages
1. Загрузите все файлы в репозиторий `ART-Sale`
2. В настройках репозитория включите GitHub Pages (ветка main, папка /)
3. Сайт будет доступен по адресу: `https://gavrosh2007.github.io/ART-Sale/`

## Для добавления иконок PWA
Положите в корень `icon-192x192.png` и `icon-512x512.png` (любые квадратные изображения)

## Для будущего API
Все данные хранятся в localStorage в формате JSON:
- `art_authors` — список авторов
- `art_works` — список работ
- `art_buy_requests` — заявки на покупку
